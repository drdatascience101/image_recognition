Python Computer Vision:

This repository contains code related to tasks in computer vision using python.

The dataset was published by Stanford and contains 20,000 images and annotations of photos of dogs that are labelled by breed. 

The images ares available in the form of `.tar` files. 
The `start.py` script contains information on how-to extract and process these files.

The `dog_keras_nn.py` script is for training a CNN with the data using keras.
